# How to run client and server:

## Server

run at `./server` folder to compile:
~~~ 
gcc -o server server.c -ljson-c
~~~~

Then, run:

~~~ 
./server
~~~~

## Client

run at `./client` folder to compile:
~~~ 
gcc -o client client.c
~~~~

Then, run to connect:

~~~ 
./client <ip_to_connect>
~~~~
